Without photos, the applications on this website are not going to give us much information
We need your help to find photos for the different projects that are currently on the website.

Finding photos is easy. All you need is an internet connection and the latest version of firefox

Step 1) Download and install the Geotag-X plugin for firefox
Step 2) Choose which project you want to work on from the list
Step 3) Start browsing for photos!

If you land on a page relevant to the project you are working on, a pop-up window will appear asking if you want to send the images to Geotag-X.
If you think they are relevant, send them! Don’t worry if you make some mistakes, we can deal with filtering later on in the Geotag-X applications.

What photos should I be sending? We are interested in any photos showing the area and time period relevant to the project. Even if you think they may not contain much useful information send them in because each project contains several different analyses on different themes and a photo that's not relevant in one application might be relevant to a different one. Even if they don't check the boxes for the questions in the applications send them in - no data can still be useful information!
